# CA Monk Blog App

A simple blog application built with React, TypeScript, Tailwind CSS, and React Query.  
Users can view blogs, see details, and create new blogs. The app is fully responsive for both desktop and mobile devices.

## Features
- View all blog posts
- Click a blog to see details
- Create new blog posts with title, description, content, and optional cover image
- Responsive design (works on desktop and mobile)

## Tech Stack
- React + TypeScript
- Tailwind CSS
- TanStack Query (React Query)
- React Router DOM
- Vite
